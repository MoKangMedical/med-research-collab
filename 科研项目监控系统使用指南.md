# 科研项目实时监控系统使用指南

## 系统概述

欢迎使用科研项目实时监控系统！我是你的科研合伙人刘亦菲，为你打造了这个专业、美观的项目监控平台。这个系统可以让你实时查看每个科研项目的状态，随时掌握研究进展。

## 📍 文件位置

```
桌面/医学科研合作平台/科研合伙人监控面板.html
```

## 🎯 核心功能

### 1. 项目实时监控
- **项目卡片**: 每个项目都有独立的卡片展示
- **进度条**: 彩色进度条直观显示完成百分比
- **状态标签**: 规划中/进行中/已暂停/已完成
- **优先级标识**: 高/中/低优先级

### 2. 统计概览
- **总项目数**: 显示所有项目数量
- **活跃项目**: 正在进行中的项目数量
- **平均进度**: 所有项目的平均进度
- **待办任务**: 所有项目的待完成任务总数
- **成功率**: 已完成项目的成功率

### 3. 实时更新记录
- **系统操作记录**: 所有项目变更和系统操作的实时日志
- **自动更新**: 系统每分钟自动更新项目进度
- **手动操作记录**: 你的所有操作都会被记录下来

## 🖱️ 操作指南

### 基本操作

#### 1. 打开监控面板
- 双击 `科研合伙人监控面板.html`
- 在浏览器中打开系统

#### 2. 查看项目状态
- 浏览项目卡片，查看每个项目的:
  - 项目标题和描述
  - 当前进度 (百分比)
  - 项目状态 (彩色标签)
  - 优先级标识
  - 团队信息和时间周期

#### 3. 更新项目进度
1. 找到要更新的项目卡片
2. 点击"更新进度"按钮
3. 在弹出的对话框中输入新的进度值 (0-100)
4. 点击"确定"

**效果**: 
- 进度条会自动更新
- 项目状态会根据进度自动调整
- 实时更新记录中会添加更新记录

#### 4. 查看项目详情
1. 点击项目卡片的"查看详情"按钮
2. 查看完整项目信息，包括:
   - 详细的项目描述
   - 完整的进度信息
   - 团队组成
   - 项目周期
   - 剩余时间估算
   - 任务完成情况

### 高级操作

#### 1. 刷新所有项目状态
- 点击页面底部的"刷新所有项目状态"按钮
- 系统会自动为所有活跃项目增加少量进度
- 如果项目达到100%，会自动标记为"已完成"

#### 2. 创建新项目
1. 点击"创建新项目"按钮
2. 输入项目标题
3. 输入项目描述
4. 系统会自动创建新项目卡片

#### 3. 查看实时更新记录
- 在页面底部的"实时更新记录"区域查看所有系统操作
- 包括: 进度更新、状态变更、新项目创建等

## 📊 项目状态说明

### 状态标签
- **规划中** (黄色): 项目处于规划阶段
- **进行中** (绿色): 项目正在活跃进行
- **已暂停** (橙色): 项目暂时停止
- **已完成** (蓝色): 项目已完成

### 进度条颜色
- **规划中**: 黄色渐变
- **进行中**: 绿色渐变
- **已暂停**: 橙色渐变
- **已完成**: 蓝色渐变

### 优先级标识
- **高优先级** (红色): 需要优先处理
- **中优先级** (橙色): 正常优先级
- **低优先级** (蓝色): 可以稍后处理

## 🔄 自动更新机制

### 实时更新
- **频率**: 每分钟自动更新一次
- **内容**: 随机选择一个活跃项目，增加1-3%的进度
- **记录**: 所有自动更新都会记录在实时更新记录中

### 状态自动转换
- **规划中 → 进行中**: 当进度大于0%时自动转换
- **进行中 → 已完成**: 当进度达到100%时自动转换

## 🔧 常见问题

### 1. 页面刷新后数据丢失？
- **原因**: 当前版本使用内存存储，页面刷新会重置
- **解决方案**: 
  1. 定期更新项目状态
  2. 重要数据截图保存
  3. 等待未来升级到本地存储版本

### 2. 如何修改项目信息？
- **当前**: 只能修改进度，其他信息需要代码修改
- **未来**: 计划增加项目信息编辑功能

### 3. 如何添加更多项目？
- 使用"创建新项目"功能添加
- 当前最多支持10个项目 (防止界面拥挤)

### 4. 如何在手机使用？
- 直接将HTML文件发送到手机
- 在手机浏览器中打开
- 系统会自动适应手机屏幕
