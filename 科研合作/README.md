## 使用指南

### 1. 新项目启动
1. 在`项目库/`中创建新文件夹：`项目库/项目名称_YYYYMMDD`
2. 复制`项目库/项目模板.md`到新项目
3. 填写项目基本信息

### 2. 文献管理
1. 在`文献库/`中按主题分类存储文献
2. 使用Zotero/EndNote管理参考文献
3. 定期更新文献综述

### 3. 数据分析
1. 原始数据存储在`数据分析/数据/`
2. R/Python脚本存储在相应目录
3. 分析报告使用Markdown格式

### 4. 论文写作
1. 使用LaTeX/Word模板
2. 图表单独存储，便于更新
3. 记录投稿状态和审稿意见

### 5. 会议记录
1. 每次讨论后及时记录
2. 记录决策和待办事项
3. 定期回顾会议记录

## 协作规范

### 文件命名规范
- 日期格式：YYYYMMDD
- 版本控制：v1.0, v1.1等
- 清晰描述：避免使用模糊名称

### 数据管理
- 原始数据永不修改
- 分析过程可重复
- 结果文件清晰标注

### 代码规范
- 添加详细注释
- 使用版本控制
- 文档与代码同步

## 快速开始

### 第一步：了解科研背景
阅读`医学科研合作框架.md`，了解合作模式。

### 第二步：创建首个项目
```bash
# 创建项目文件夹
cp -r 项目库/项目模板.md 项目库/我的首个项目_20260222/
```

### 第三步：开始工作
根据项目需求，在相应目录中开展工作。
